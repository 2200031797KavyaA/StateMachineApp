- همچنین برای تبدل عبارت NFA به Reduction کافیست از مسیر /reduction/ عبارت NFA را به صورت جیسون در URL اضافه کرده و خروجی Reduction خود را به صورت خروجی جیسون دریافت کنید

Welcome to Atamata


Atamata web service is here where you can use it to do your machine calculations

- To convert NFA to DFA, you just add NFA as Jason in the URL from the /nfa-to-dfa/ path and get your DFA output as Jason output.

- Also, to convert NFA to Reduction, just add NFA as Jason in the URL from the /reduction/ path and get your Reduction output as Jason output.

Quick start

nfa to dfa

1. Load https://automaton-app.herokuapp.com/nfa-to-dfa/{your json form of nfa}

2. Get the first Array object of the JSON response

3. Load it’s .url

4. Enjoy your dfa as json form

Reduction

1. Load https://automaton-app.herokuapp.com/reduction/{your json form}

2. Get the first Array object of the JSON response

3. Load it’s .url

4. Enjoy your Reduction as json form

Error handling

- If use DFA form as input you get json which has ‘is_dfa’ Key that validate by True.

- If program get any errors and it don’t work and it gets ‘status’: null .
